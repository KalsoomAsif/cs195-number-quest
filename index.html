Number Quest

Guess the secret number (1–100) before you run out of attempts.

About the Game

A simple, fast number guessing game built with semantic HTML, CSS Grid/Flexbox, and vanilla JavaScript. It features color‑coded feedback, a guess history, difficulty levels, and a saved best score using localStorage.

  • Clear win/lose conditions
  • Responsive design (phone + desktop)
  • “Learn Something New” feature: saved Best Score
